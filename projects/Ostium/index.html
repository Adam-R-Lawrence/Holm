Ostium

Ostium is a geospatial processing library for converting terrain, satellite, and hydrographic data into simulation-ready inputs for Torrentem.

Status
Research tooling
Inputs
SRTM, Sentinel, GIS layers, river networks
Outputs
Domains, boundaries, mesh fields, hydraulic parameters

Purpose

Keep environmental data preparation close to the numerical model so terrain, boundary, and hydraulic assumptions can be reproduced and revised.

Methods and Tools

  • GIS and remote-sensing data normalization for terrain and channel information.
  • Domain preparation for Torrentem meshes and boundary metadata.
  • Parameter extraction for roughness, slope, extents, and hydrographic context.

Current Limitations

Workflows remain tied to research cases and data availability; interfaces are expected to change as Torrentem input formats mature.

Related Tools

Ostium sits upstream of Torrentem. Vadum and downstream notebooks can then inspect the simulation fields produced from those prepared domains.
