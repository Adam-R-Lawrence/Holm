Stratum

Stratum is a build-preparation and G-code generation toolkit for DLP and SLA photopolymerization printers, tuned to consume volumetric exposure plans exported from Torrentem simulations.

Status
Prototype toolchain
Core tasks
Slicing, exposure planning, lift trajectory optimization
Machines
DLP and SLA resin printers

Purpose

Translate photopolymerization simulation outputs into machine-aware layer plans and printer instructions that can be checked before fabrication.

Methods and Tools

  • Layer slicing from simulated exposure plans.
  • G-code generation for DLP and SLA resin-printer workflows.
  • Build profiles that account for exposure, lift motion, and printer constraints.

Current Limitations

The toolchain is prototype-stage and depends on calibration data, printer-specific assumptions, and the current Torrentem photopolymerization output format.

Related Tools

Stratum sits downstream of Torrentem's additive-manufacturing simulations and is intended to connect numerical exposure studies with practical build preparation.
